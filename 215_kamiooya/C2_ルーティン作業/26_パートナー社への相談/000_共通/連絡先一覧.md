## 補足

- **メール**: Gmail API で届いたメールの送信元と照合し、該当フォルダの `やり取り.md` に自動追記する際に使用。
- **電話**: iMessage/SMS 連携で電話番号を照合し、該当フォルダの `やり取り.md` に自動追記する際に使用。
- **Chatwork**: Chatwork API 連携で `room_id` と相手名を照合し、該当フォルダの `やり取り.md` に自動追記する際に使用。
- **LINE**: 手動判別用。返信が LINE で来たとき、この一覧を見て該当フォルダを特定してから `やり取り.md` にコピペする。
