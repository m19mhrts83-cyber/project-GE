## 📊 Excelシートを指定して送信

Excelファイルに複数のシートがある場合、特定のシートを指定できます。

### 1️⃣ 管理会社Aのシートに送信

```
@空室対策_Grandole志賀本通2_260129.md と @送信先一覧.xlsx のシート「管理会社A」を使ってメール送信（自動承認）
```

### 2️⃣ 管理会社Bのシートに送信

```
@空室対策_Grandole志賀本通2_260129.md と @送信先一覧.xlsx のシート「管理会社B」を使ってメール送信（自動承認）
```

### 3️⃣ 全管理会社のシートに送信

```
@空室対策_Grandole志賀本通2_260129.md と @送信先一覧.xlsx のシート「全管理会社」を使ってメール送信（自動承認）
```

### 💡 シート名の指定方法

チャットで以下のように指定してください：
- `のシート「シート名」を使って` の形式
- シート名は実際のExcelファイル内のシート名と完全一致させる

**Excelファイルのシート例：**
```
送信先一覧.xlsx
├─ 管理会社A
├─ 管理会社B
├─ 全管理会社
└─ テスト送信
```

## 📂 ファイル名のパターン

### MDファイル（メール本文）のパターン

- `空室対策_[物件名]_[日付].md`
- 例：`空室対策_Grandole志賀本通2_260129.md`

### Excelファイル（送信先リスト）のパターン
★管理会社一覧.xlsx
シート「G2」

#### テスト送信用
- `テスト送信リスト.xlsx`（自分宛てのみ）

#### 本番送信用
- `★管理会社一覧.xlsx`（不動産会社リスト）
- シート「G2」
- /Users/matsunomasaharu/Library/CloudStorage/OneDrive-個人用/215_神・大家さん倶楽部/20_【空室対策】【修繕】【売却】/21_【空室対策】募集,ステージング,物件管理/★管理会社一覧.xlsx

### Excelファイルのシート構成例

**例1：エリア別に分ける**
```
送信先一覧.xlsx
├─ 名古屋エリア（名古屋市内の管理会社）
├─ 東京エリア（東京都内の管理会社）
├─ 大阪エリア（大阪市内の管理会社）
└─ 全管理会社（全エリアまとめて送信）
```

**例2：管理会社別に分ける**
```
送信先一覧.xlsx
├─ A社
├─ B社
├─ C社
└─ 全管理会社
```

**例3：物件別に分ける**
```
送信先一覧.xlsx
├─ Grandole志賀本通2担当
├─ ◯◯マンション担当
├─ △△アパート担当
└─ 全管理会社
```

### シート内の列構成

各シートには以下の列を用意してください：

| メールアドレス | 会社名 | 担当者名 | 備考 |
|---|---|---|---|
| example1@company.com | A不動産 | 山田太郎 | 名古屋エリア担当 |
| example2@company.com | B不動産 | 佐藤花子 | 東京エリア担当 |

**必須列：** `メールアドレス`（この列名は必ず必要です）
**任意列：** その他の列（会社名、担当者名など）は管理用に自由に追加可能

## 🔧 トラブルシューティング

### エラーが出た場合

#### 1. ファイルが見つからない

**原因：** ファイル名が間違っているか、パスが正しくない

**解決策：**
- ファイル名を正確にコピー＆ペースト
- Cursorの `@` メンション機能を使って正確に指定

#### 2. 認証エラーが出た場合

**原因：** 認証トークンの有効期限切れ

**解決策：**
```
token.pickleを削除して再認証してください
```

## 📝 よくある使い方

### パターンA：新しい物件の空室対策メール

1. 新しいMDファイルを作成（例：`空室対策_新物件名_260201.md`）
2. 1行目に件名、2行目以降に本文を記載
3. Cursorチャットで以下を実行：

```
@260129_G2_空室対策.md と @テスト送信リスト.xlsx を使ってメール送信（自動承認）
```

4. 自分宛てに届いたメールを確認
5. 問題なければ本番送信：

```
@空室対策_新物件名_260201.md と @C1_cursor/mail_automation/送信先一覧.xlsx を使ってメール送信（自動承認）
```

### パターンB：定期的な空室状況報告

毎月の空室状況を報告する場合：

1. 前回のMDファイルをコピーして日付を更新
2. 内容を最新情報に更新
3. 時間指定で送信：

```
@空室対策_月次報告_260201.md と @C1_cursor/mail_automation/送信先一覧.xlsx を使って10:00にメール送信
```

### パターンC：特定の管理会社グループにのみ送信

管理会社をグループ分けしている場合：

1. Excelファイルで管理会社ごとにシートを分ける
   - 例：「名古屋エリア」「東京エリア」「全管理会社」

2. 送信したいグループのシートを指定：

```
@空室対策_Grandole志賀本通2_260201.md と @送信先一覧.xlsx のシート「名古屋エリア」を使ってメール送信（自動承認）
```

3. 複数のエリアに順次送信する場合は、シートごとに実行：

```
@空室対策_Grandole志賀本通2_260201.md と @送信先一覧.xlsx のシート「名古屋エリア」を使ってメール送信（自動承認）
```

その後：

```
@空室対策_Grandole志賀本通2_260201.md と @送信先一覧.xlsx のシート「東京エリア」を使ってメール送信（自動承認）
```

### パターンD：A社とB社で異なる内容を送信

管理会社ごとに異なる内容を送る場合：

1. 管理会社ごとにMDファイルを作成
   - `空室対策_A社向け_260201.md`
   - `空室対策_B社向け_260201.md`

2. それぞれのシートに送信：

```
@空室対策_A社向け_260201.md と @送信先一覧.xlsx のシート「A社」を使ってメール送信（自動承認）
```

```
@空室対策_B社向け_260201.md と @送信先一覧.xlsx のシート「B社」を使ってメール送信（自動承認）
```

## 💡 便利なTips

### Tip 1：@メンション機能を活用

Cursorで `@` を入力すると、ファイル候補が表示されます。
ファイル名を手入力せず、候補から選択することでタイポを防げます。

### Tip 2：テンプレートMDファイルを作成

よく使う文面のテンプレートを作成しておくと便利です：
- `空室対策_テンプレート.md`
- `入居お礼_テンプレート.md`

### Tip 3：送信履歴の確認

送信履歴は以下に自動保存されます：
```
C1_cursor/mail_automation/logs/send_history.log
```
