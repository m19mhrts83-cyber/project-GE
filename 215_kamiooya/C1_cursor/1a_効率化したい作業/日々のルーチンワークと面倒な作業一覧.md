## 1. 現在のルーチンワーク（定期的にやっている作業）

### 1.1 空室対策メールの送信（週〜月次）

| 項目 | 内容 |
|------|------|
| **頻度** | 空室発生時・継続募集時（例：1/22, 1/31, 2/5, 2/13 など） |
| **手順** | 1. MDファイルで件名・本文を用意 2. テスト送信 3. 本番送信 |
| **面倒ポイント** | Cursorのターミナルでは動かないため、**Cursorを終了してから** AppleScript（.scpt）またはMacターミナルで実行する必要がある |
| **関連** | `C2_ルーティン作業/24_空室対策メール履歴/`、`送信実行手順.md` |

### 1.2 パートナー社（管理会社・修繕会社）とのやり取り管理

| 項目 | 内容 |
|------|------|
| **対象** | ホームプランナー、ミニテック、Tcell、LEAF（管理会社）、立木（修繕会社） |
| **手順** | 1. 相手から返信が来る 2. やり取り.md にコピペ 3. 返信案を考える（Watsonに依頼可） 4. 送信下書き.txt に貼る 5. メールに貼って送信 6. 送信後、やり取り.md に履歴追記 |
| **自動化の有無** | Gmail → やり取り.md への自動追記あり（`gmail_to_yoritoori.py`）。LINE・iMessageは手動コピペ |
| **関連** | `C2_ルーティン作業/26_パートナー社への相談/`、`管理会社メール履歴_運用の選び方.md` |
