📖 使い方

  1. Google Maps API Keyを取得
    • Google Cloud Consoleにアクセス
    • プロジェクトを作成または選択
    • 「APIとサービス」→「ライブラリ」から以下を有効化：
      • Maps JavaScript API
      • Places API
      • Geocoding API
    • 「APIとサービス」→「認証情報」からAPI Keyを作成
    • （推奨）API Keyに制限を設定：
      • アプリケーションの制限: HTTPリファラー
      • APIの制限: 上記3つのAPIのみ
  2. API Keyを入力して保存（初回のみ）
  3. 駅名を入力（例：渋谷駅、新宿駅、東京駅）
  4. 検索ボタンをクリックして結果を表示
  5. 表形式に切り替えて「📋 表をコピー」でExcelへ貼り付け

💡 コスト削減のヒント

  • 🔍 検索結果をメモ：同じ駅を何度も検索しない

  • ⚙️ 予算アラートを設定：一定金額を超えたら通知を受け取る
