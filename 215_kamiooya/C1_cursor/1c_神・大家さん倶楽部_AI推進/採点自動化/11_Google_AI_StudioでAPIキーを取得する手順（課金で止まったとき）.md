# Google AI Studio で API キーを取得する手順（有料プラン・課金設定）

GAS（Google Apps Script）でこの採点ツールの **Gemini 判定** を使うには、**有料プランで課金設定（クレジットカード登録）を完了した状態** が必要です。  
無料枠では判定ができず、情報確認の結果も「有料プラン＋課金設定を行わないと進めない」とのことです。

このドキュメントは、**課金設定を早く通して先に進む** ための具体的な手段をまとめています。

---

## 前提：有料プラン＋課金設定が必須

- **無料のままだと Gemini の判定ができない** ため、この採点ツールでは有料プランが前提です。
- **クレジットカードを登録し、課金が有効になった状態** にしないと先に進めません。
- 以下は、その課金設定を **早く完了させる** ための方法です。

---

## 課金設定を早く通すための手段

### 1. 請求先アカウントを「新規作成」し直す（おすすめ）

いまのプロジェクトで審査が止まっている場合は、**新しい請求先アカウント** を作成し、**新しいプロジェクト** にリンクすると、先に進むことがあります。

1. **Google Cloud コンソール** で **https://console.cloud.google.com/billing/create** を開く（または ナビゲーション → 「お支払い」→ 請求先アカウントを作成）。
2. **国**（日本）と **請求先アカウント名** を入力し、**クレジットカード** を登録する。
3. 作成した請求先アカウントを、**新しいプロジェクト** にリンクする。  
   - コンソール上部のプロジェクト選択 → 「新しいプロジェクト」で作成 → そのプロジェクトの「お支払い」で、今つくった請求先アカウントをリンク。
4. そのプロジェクトで **Google AI Studio** から「API キーを作成」→ そのプロジェクトを選んでキーを発行する。
5. 発行した API キーを GAS のスクリプト プロパティ `GEMINI_API_KEY` に設定する。

**ポイント**：クレジットカードは **銀行口座より確認が早い** です。銀行口座だと最大10日かかることがあります。

---

### 2. クレジットカードの名義・住所を Google アカウントと揃える

審査で止まりやすい原因の一つが、**登録情報の不一致** です。

- **カードの名義**（漢字・ローマ字）が、Google アカウントの名前と一致していると通りやすいです。
- **住所** も、Google アカウントや請求先アカウントで入力した住所と揃えておくとよいです。
- 不一致や不鮮明な書類で身分証明を求められた場合、拒否されると理由が明示されないことがあります。

---

### 3. 別の支払い方法を試す（バーチャルカードなど）

- 同じカードで何度も失敗している場合は、**別のクレジットカード** や **バーチャルカード**（例：Revolut などで発行）を登録し直すと、数時間で審査が通ったという報告があります。
- 利用する場合は、サービス利用規約と手数料を確認してください。

---

### 4. 課金が「有効」になっているか確認する

クレジットカードを登録しただけでは不十分で、**プロジェクトに請求先アカウントがリンクされ、課金が有効** になっている必要があります。

1. **Google Cloud コンソール** → 使っている **プロジェクト** を選択。
2. 左メニュー **「お支払い」**（Billing）を開く。
3. そのプロジェクトに **請求先アカウントがリンク** されており、ステータスが **「アクティブ」** か確認する。
4. リンクされていない場合は、「請求先アカウントをリンク」から、カードを登録済みの請求先アカウントを選んでリンクする。

ここまでできていれば、**Google AI Studio** でそのプロジェクトを選んで API キーを作成し、GAS で使えます。

---

### 5. 審査が長引く場合の目安とサポート

- クレジットカード登録後、**数時間〜2 日程度** かかることがあります。
- **銀行口座** の場合は、確認に **最大10日** かかることがあります（クレジットカードの方が早いです）。
- それ以上かかる、または「お支払い方法を確認できませんでした」などのメールが届いた場合は、**Google Cloud Billing のヘルプ** から「問題を解決する」や **サポートへの問い合わせ** を検討してください。  
  - ドキュメント: [Cloud Billing の問題を解決する](https://cloud.google.com/billing/docs/how-to/resolve-issues?hl=ja)

---

## 課金が有効になったあとにすること

1. **Google AI Studio**（https://aistudio.google.com/）を開く。
2. **「API キー」** → **「API キーを作成」** をクリック。
3. **課金を有効にしたプロジェクト** を選んでキーを発行する（既存プロジェクトを選ぶか、AI Studio 側で「新しいプロジェクトを作成」し、そのプロジェクトが請求先にリンクされていることを確認）。
4. 表示された **API キーをコピー** し、GAS の **スクリプト プロパティ** に `GEMINI_API_KEY` として保存する（`07_超初心者_GAS組み込み実作業マニュアル.md` 参照）。

これで有料枠の Gemini が GAS から使え、採点ツールの判定が動く状態になります。

---

## まとめ

| 状況 | やること |
|------|----------|
| 課金設定で止まっている | **請求先アカウントを新規作成** → **新しいプロジェクト** にリンク → クレジットカードで登録（銀行より早い） |
| 審査が通らない | カードの**名義・住所を Google アカウントと一致**させる／**別のカード・バーチャルカード**を試す |
| 登録したが動かない | **お支払い**画面でプロジェクトに請求先が**リンク**され、**アクティブ**か確認する |
| 長く待っても通らない | **Billing のヘルプ・サポート**に問い合わせる |

**無料のままだと判定できない** ため、**有料プランで課金設定を完了させる** ことが先に進む条件です。上記の手順で、できるだけ早く課金を有効化し、API キーを取得してください。
