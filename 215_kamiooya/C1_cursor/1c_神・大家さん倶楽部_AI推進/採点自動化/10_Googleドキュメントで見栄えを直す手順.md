# Google ドキュメントで「見栄え」を直す手順（超初心者向け）

Markdown をコピペしただけだと、`##` や `-` がそのまま表示されたり、**背景色（ハイライト）が付いたまま**になったりして読みづらいです。  
以下の手順で、背景色を消し、見出し・箇条書き・番号リストに整えます。

---

## 貼り付けた直後：背景色（ハイライト）を消す

コピペすると、もとのアプリの背景色や蛍光ペンがそのまま付いてくることがあります。**まず全体の背景を「なし」にすると、あとの作業がしやすくなります。**

1. ドキュメント内の**どこかでクリック**してカーソルを置きます。
2. **Ctrl + A**（Mac: **Cmd + A**）で**全文を選択**します。
3. 画面上部のツールバーで **「蛍光ペン」アイコン**（マーカーっぽいマーク）を探してクリックします。  
   （「A」の文字の色の右隣あたりにあります。「▼」が付いたプルダウンになっています。）
4. 一覧のいちばん上にある **「なし」** を選びます。

これで全文の背景色（ハイライト）が消えます。選択を解除するには、文のどこかをクリックすればOKです。

---

## 手順0：検索と置換で記号を一括削除（効率化）

**先に「検索と置換」で Markdown の記号を消しておくと、あとで見出し・リストを付けるだけになって楽です。**

### 検索と置換の開き方

- メニュー **「編集」→「検索と置換」**（Windows: **Ctrl + H** / Mac: **Cmd + H**）

### 置換の順序（この順で実行する）

| 順番 | 検索する文字 | 置換後 | 操作 |
|------|----------------|--------|------|
| 1 | `## `（## と半角スペース） | **何も入れない（空欄）** | 「すべて置換」 |
| 2 | `### `（### と半角スペース） | **何も入れない** | 「すべて置換」 |
| 3 | `---` | **何も入れない** | 「すべて置換」 |

- **「1. 」の前に**、いちばん上の `# 神大家さんクラブ向け...` の **`# ` だけ手動で削除**しておく（検索では「# 」を全部消すと、見出しの「#」以外の場所が壊れる可能性があるため、タイトル行だけ手で消すのが安全です）。

### 注意：「- 」と「1. 」は一括置換しない

- **`- `（ハイフン＋スペース）**  
  本文中の「〇〇 - 〇〇」も消えてしまうので、**一括置換はしない**。手順3のとおり、箇条書きにしたい行だけ手で消す。
- **`1. `「2. 」**  
  見出しの「1. 全体像」の「1. 」まで消えるので、**一括置換はしない**。手順4のとおり、番号リストにしたいブロックだけ手で消す。

### 置換が終わったあと

- 記号（`##` `###` `---`）が消えているので、**手順1〜4では「行を選んで書式（見出し・箇条書き・番号リスト）を付けるだけ**でよくなります。**

## 準備：Google ドキュメントを開いておく

- すでに「神大家さんクラブ向け WeStudy成果報告AI採点ツール 概要」のドキュメントを開いている状態を想定しています。
- これから「該当する行を選んで、書式を変える」作業をしていきます。

## 手順1：「見出し」にしたい行を直す

「見出し」にすると、文字が大きく・太くなり、左の「ドキュメントのアウトライン」にも表示されます。

### 1-1. `# 神大家さんクラブ向け...` の行（いちばん上のタイトル）

1. 手順0で **`# ` だけは手動で削除**しておく（検索では消さない想定）。
2. その行の**左端（余白）をクリック**して、行全体を選択します。
3. 画面上部のプルダウン **「標準のテキスト」** をクリック → **「タイトル」** を選びます。

これで一番上のタイトルが「タイトル」の書式になります。

#### 「左端をクリック」ってどこ？（添付図で説明）

**「左端」は、画面上のハンバーガーメニュー（≡）や、左の「ドキュメント タブ」のサイドバーではありません。**

- **クリックする場所**：**本文が表示されている白いエリア**のなかで、**変更したい行の、文字の左側にある余白**です。
- つまり、「神大家さんクラブ向け…」や「## 1. 全体像…」などの**行の、左端の余白（文字が始まるより左の、何もないスペース）**を1回クリックすると、その行全体が選択されます。
- マウスをその余白に持っていくと、矢印が**右上向きの白い矢印**に変わるので、その状態でクリックすると「行選択」になります。

![「左端」のイメージ：本文エリアの、行の左側の余白をクリックする](image/09_目黒さん向け_成果報告AI採点ツール概要/1773360098229.png)

※ 図では、本文の左側に「ドキュメント タブ」のパネルが出ていますが、**クリックするのはそのパネルではなく、本文（白い編集領域）の、各行の左にある余白**です。

### 1-2. `## 1. 全体像...` のような行（大見出し）

手順0で `## ` はすでに消えているので、**書式を付けるだけ**です。

1. `1. 全体像（何ができるか）` の行の**左端をクリック**して行選択。
2. プルダウン「標準のテキスト」をクリック → **「見出し 1」** を選びます。

同様に、次の行も「見出し 1」にします。

- `2. 関係ファイル一覧...`
- `3. 目黒さん側で「必ず変更が必要な箇所」`
- `4. Google Cloud 側の前提条件...`
- `5. 月次運用フロー...`
- `6. 今後の拡張の方向性...`

**コツ**：行の左端（本文エリアの、その行の文字より左の余白）をクリックすると「その行だけ」選択できます。どこをクリックするかは、上の **「左端をクリックってどこ？」** を参照してください。

### 1-3. `### 目的` や `### 処理の流れ...` のような行（中見出し）

手順0で `### ` はすでに消えているので、**書式を付けるだけ**です。

1. `目的` の行を左端クリックで選択。
2. プルダウンから **「見出し 2」** を選びます。

同じ要領で、次のような行も「見出し 2」にします。

- `処理の流れ（ざっくり）`
- `入出力のイメージ`
- `2-1. メインスプレッドシート`
- `2-2. マニュアル類...`
- `2-3. 技術的な補足資料...`
- `3-1. Gemini API キー...`
- `3-2. Drive 上の CSV ファイル ID...`
- `3-3. 実行パラメータ...`
- `必須条件`
- `もし limit がすべて 0 の場合`

**「見出し 1」＝大見出し、「見出し 2」＝その下の中見出し**と覚えておけばOKです。

## 手順2：`---` の行を整理する

手順0で「---」を**空欄に置換**していれば、その行は空白行になっています。

- **A）そのまま**  
  空白行として区切りに使う。
- **B）横線にしたい**  
  その行にカーソルを置き、メニュー **「挿入」→「水平線」** で線を入れてから、空白行を削除する。

お好みで A か B を選んでください。（手順0で「---」を置換していない場合は、`---` の行を選択して Delete で消すか、水平線に差し替えてください。）

---

## 手順3：「箇条書き」（`-` で始まる行）を整える

1. **最初の `-` で始まる行**（例：`- WeStudyの「成果報告コミュニティ」の…`）の**行頭をクリック**してカーソルを置きます。
2. 画面上部のツールバーで **「箇条書き」のアイコン**（・が並んだマーク）をクリックします。
3. その行の先頭の `- ` を**手で削除**します。
4. 次の行も `-` で始まっていれば、同じ行の**末尾で Enter** を押すと、次の行が自動で箇条書きになります。その行の `- ` も削除します。
5. 箇条書きにしたくない行（普通の段落）が来たら、その行で **箇条書きアイコンをもう一度クリック**してオフにするか、**BackSpace でインデントを戻す**と、通常の段落に戻ります。

**まとめ**：`-` で始まる行は「箇条書き」を付けてから `-` を消す、と覚えておけば大丈夫です。

---

## 手順4：「番号付きリスト」（1. 2. 3.）を整える

「処理の流れ」の 1. 2. 3. 4. 5. のような部分です。

1. **「1. 管理画面から…」** の行の行頭にカーソルを置きます。
2. ツールバーの **「番号付きリスト」のアイコン**（1. 2. 3. のマーク）をクリックします。
3. その行の **「1. 」を手で削除**します。
4. 次の行「2. スプレッドシートの…」も、**Enter で続けて番号付きリストにし、「2. 」を削除**します。
5. 3. 4. 5. も同様に、番号付きリストにしてから先頭の「3. 」などを削除します。
6. リストが終わる行の次では、**Enter を2回押す**か、**番号付きリストアイコンをもう一度クリック**してリストを終了します。

---

## 手順5：表（テーブル）を整える

Markdown の表は、Google ドキュメントでは「表」として作り直すと見やすくなります。

### やり方（1つ目の表の例）

1. 表にしたい部分（例：「入力」「中間」「出力」のブロック）を**ドラッグで選択**します。
2. メニュー **「挿入」→「表」** を選び、マス目を選んで表を挿入します（例：3列×2行）。
3. 上段のセルに「項目」「内容」、下の行に「入力」「WeStudy CSV…」のように、**もとのテキストをコピーしてセルに貼り付け**ます。
4. もとの Markdown の表の行（`| 項目 | 内容 |` や `|------|------|` など）は**削除**します。

表がたくさんある場合は、1つずつ同じ手順で「挿入→表」で作ってから、中身を写していくとよいです。

**コツ**：表にしなくても、該当部分を「箇条書き」や「改行で区切った段落」のままにしておいても読めます。無理に表にしなくて大丈夫です。

---

## 手順6：フォントと余白（任意）

- **フォント**  
  メニュー **「書式」→「段落のスタイル」→「標準のテキスト」** などで、お好みのフォント（例：游ゴシック、Noto Sans JP）に変更できます。  
  全体を選択（Ctrl+A / Cmd+A）してから、ツールバーのフォント名で選べば一括変更できます。
- **余白**  
  **「ファイル」→「ページ設定」** で、余白を少し広げると読みやすくなります。
