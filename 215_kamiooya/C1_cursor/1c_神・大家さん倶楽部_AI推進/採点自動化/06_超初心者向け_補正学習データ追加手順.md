# 超初心者向け: 補正学習データを追加する手順

この手順は、以下を目的にしています。

- Geminiの採点結果を人間が補正した内容を記録する
- 記録した補正内容を次回採点の参考に使える状態にする

---

## 0. 先に理解しておくこと（30秒）

- `採点結果` シート: Geminiが出した結果が入る場所
- `手動補正点` 列: 人間が修正点を入れる場所
- `補正学習データ` シート: 補正履歴をためる場所（今回新しく作る）

---

## 1. スプレッドシート側の準備（画面操作）

## 1-1. まずスプレッドシートを開く

1. 採点に使っているGoogleスプレッドシートを開く
2. 下のタブに `採点結果` があることを確認

## 1-2. 新しいシートを作る

1. 左下の `+`（シート追加）をクリック
2. シート名を `補正学習データ` に変更

## 1-3. 見出し（ヘッダー）を入力

`補正学習データ` の1行目に次を左から順に入力してください。

| 列 | 見出し |
|---|---|
| A | 追加日時 |
| B | コメントID |
| C | 投稿者名 |
| D | コメント内容 |
| E | Gemini得点 |
| F | 手動補正点 |
| G | 点差 |
| H | GeminiルールID |
| I | 最終ルールID |
| J | 補正理由メモ |
| K | 採点バージョン |

---

## 2. GAS（Apps Script）に関数を追加する

## 2-1. Apps Scriptを開く

1. スプレッドシート上部メニュー `拡張機能`
2. `Apps Script` をクリック

## 2-2. 既存の `Code.gs` を開く

- 前回追加済みの `04_gas_Code.gs` 相当のコードが入っているファイルです。

## 2-3. 以下のコードを一番下に貼り付け

```javascript
function appendCorrectionLearningData() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var resultSheet = ss.getSheetByName("採点結果");
  var learnSheet = ss.getSheetByName("補正学習データ");

  if (!resultSheet) throw new Error("採点結果シートがありません。");
  if (!learnSheet) throw new Error("補正学習データシートがありません。");

  var resultValues = resultSheet.getDataRange().getValues();
  if (resultValues.length <= 1) return;

  // 採点結果ヘッダー位置
  var headers = resultValues[0];
  var idx = {};
  for (var i = 0; i < headers.length; i++) {
    idx[String(headers[i]).trim()] = i;
  }

  var needCols = [
    "コメントID", "投稿者名", "コメント内容", "得点", "手動補正点", "ルールID", "採点バージョン"
  ];
  for (var n = 0; n < needCols.length; n++) {
    if (idx[needCols[n]] === undefined) {
      throw new Error("採点結果シートに必要列がありません: " + needCols[n]);
    }
  }

  // すでに学習データに入れたコメントIDを収集（重複追加防止）
  var existing = {};
  var lv = learnSheet.getDataRange().getValues();
  if (lv.length > 1) {
    for (var r = 1; r < lv.length; r++) {
      var cid = lv[r][1]; // B列: コメントID
      if (cid !== "" && cid !== null) existing[String(cid)] = true;
    }
  }

  var appendRows = [];
  for (var r2 = 1; r2 < resultValues.length; r2++) {
    var row = resultValues[r2];
    var commentId = row[idx["コメントID"]];
    var authorName = row[idx["投稿者名"]];
    var body = row[idx["コメント内容"]];
    var aiScore = row[idx["得点"]];
    var manualScore = row[idx["手動補正点"]];
    var ruleId = row[idx["ルールID"]];
    var version = row[idx["採点バージョン"]];

    // 手動補正がある行のみ対象
    if (manualScore === "" || manualScore === null) continue;
    if (!commentId) continue;
    if (existing[String(commentId)]) continue;

    var diff = "";
    if (aiScore !== "" && aiScore !== null && !isNaN(aiScore) && !isNaN(manualScore)) {
      diff = Number(manualScore) - Number(aiScore);
    }

    appendRows.push([
      new Date(),         // A: 追加日時
      commentId,          // B: コメントID
      authorName,         // C: 投稿者名
      body,               // D: コメント内容
      aiScore,            // E: Gemini得点
      manualScore,        // F: 手動補正点
      diff,               // G: 点差
      ruleId,             // H: GeminiルールID
      "",                 // I: 最終ルールID（手入力）
      "",                 // J: 補正理由メモ（手入力）
      version             // K: 採点バージョン
    ]);
  }

  if (appendRows.length > 0) {
    learnSheet
      .getRange(learnSheet.getLastRow() + 1, 1, appendRows.length, appendRows[0].length)
      .setValues(appendRows);
  }

  SpreadsheetApp.getUi().alert("補正学習データに " + appendRows.length + " 件追加しました。");
}
```

## 2-4. 保存する

1. 左上の保存アイコンをクリック
2. エラーが出なければOK

---

## 3. メニューにボタンを追加（任意だけど推奨）

すでに `onOpen()` がある場合、次の1行を追加してください。

```javascript
.addItem("補正学習データを追加", "appendCorrectionLearningData")
```

`onOpen()` 例:

```javascript
function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu("採点自動化")
    .addItem("CSV取込（Drive File ID）", "importCsvByConfig")
    .addItem("採点実行", "runScoring")
    .addItem("補正学習データを追加", "appendCorrectionLearningData")
    .addItem("ヘッダー初期化", "initializeSheets")
    .addToUi();
}
```

保存したら、スプレッドシートを再読み込みしてください。

---

## 4. 実際の運用手順（月次）

1. `採点実行` を行う
2. `採点結果` シートで必要な行だけ `手動補正点` を入力
3. メニュー `採点自動化 > 補正学習データを追加` を実行
4. `補正学習データ` シートに追加された内容を確認
5. `最終ルールID` と `補正理由メモ` を必要な分だけ手入力

---

## 5. よくあるつまずき

## 5-1. 追加件数が0件になる

- `採点結果` の `手動補正点` が空欄の可能性があります。
- すでに同じ `コメントID` が `補正学習データ` にあると再追加されません。

## 5-2. 「必要列がありません」エラー

- `採点結果` のヘッダー名がコード中の列名と一致しているか確認してください。
  - `コメントID`
  - `投稿者名`
  - `コメント内容`
  - `得点`
  - `手動補正点`
  - `ルールID`
  - `採点バージョン`

## 5-3. 権限エラー

- 初回実行時はGoogleの権限許可が必要です。
- 画面の案内に従って許可してください。
