# メール送信システム クイックガイド

## 🎯 よく使う操作

### 1. 基本的な送信（プレビュー→確認→送信）

```bash
cd "/Users/matsunomasaharu/Library/CloudStorage/OneDrive-個人用/215_神・大家さん倶楽部/C1_cursor/mail_automation"
source venv/bin/activate

python send_mail.py \
  --md-file "../C2_ルーティン作業/24_空室対策メール履歴/空室対策_Grandole志賀本通2_260129.md" \
  --excel-file "../送信先メールアドレス一覧.xlsx"
```

**実行の流れ：**
1. MDファイルとExcelファイルを読み込み
2. プレビューを表示（件名、本文の先頭300文字、送信先10件）
3. `yes` と入力して確認
4. メール送信完了

---

### 2. 本文全文を確認してから送信

```bash
python send_mail.py \
  --md-file "../C2_ルーティン作業/24_空室対策メール履歴/空室対策_Grandole志賀本通2_260129.md" \
  --excel-file "../送信先メールアドレス一覧.xlsx" \
  --full-preview
```

**違い：** 本文が300文字以上ある場合、全文が表示されます。

---

### 3. 送信時刻を指定（スケジュール送信）

#### パターンA: 今日の14時30分に送信

```bash
python send_mail.py \
  --md-file "../C2_ルーティン作業/24_空室対策メール履歴/空室対策_Grandole志賀本通2_260129.md" \
  --excel-file "../送信先メールアドレス一覧.xlsx" \
  --schedule "14:30"
```

**実行の流れ：**
1. プレビュー表示
2. 送信予定時刻の確認
3. `yes` と入力
4. 指定時刻まで待機（残り時間が表示されます）
5. 指定時刻になったら自動送信

**注意：** 待機中にターミナルを閉じると送信されません。待機中は `Ctrl+C` でキャンセル可能。

#### パターンB: 日付と時刻を指定

```bash

  --md-file "../C2_ルーティン作業/24_空室対策メール履歴/空室対策_Grandole志賀本通2_260129.md" \
  --excel-file "../送信先メールアドレス一覧.xlsx" \
  --schedule "2026-02-01 09:00"
```

#### パターンC: 30分後に送信

```bash
python send_mail.py \
  --md-file "../C2_ルーティン作業/24_空室対策メール履歴/空室対策_Grandole志賀本通2_260129.md" \
  --excel-file "../送信先メールアドレス一覧.xlsx" \
  --schedule "30分後"
```

### 4. 送信せずにプレビューだけ確認

```bash

  --md-file "../C2_ルーティン作業/24_空室対策メール履歴/空室対策_Grandole志賀本通2_260129.md" \
```

1. プレビュー表示
2. `[DRY-RUN モード] 実際の送信は行いませんでした` と表示されて終了

**使い道：**
- 本番送信前の最終確認
- メール内容のチェック
- 送信先リストの確認

## ⚠️ 注意事項

### スケジュール送信について

1. **ターミナルを閉じない**: 待機中にターミナルを閉じると送信されません
2. **PCをスリープしない**: スリープすると送信が失敗する可能性があります
3. **長時間の待機**: 数時間以上先の場合は、cron や macOS の「カレンダー」アプリでのスケジューリングを検討してください

### 本番送信前のチェックリスト

- [ ] MDファイルの内容は最新か？
- [ ] 件名は正しいか？
- [ ] 本文に誤字脱字はないか？
- [ ] 送信先リストは最新か？
- [ ] 送信先件数は適切か？
- [ ] テスト送信は完了しているか？

## 🔧 トラブルシューティング

### Q: プレビューで本文が途中で切れている

**A**: デフォルトでは先頭300文字のみ表示されます。全文を見るには `--full-preview` を追加してください。

### Q: スケジュール送信で待機中にエラーが出た

**A**: 以下を確認してください：
- ターミナルを閉じていないか
- PCがスリープしていないか
- インターネット接続は安定しているか

### Q: 送信時刻の指定がエラーになる

**A**: 以下のフォーマットを使用してください：
- `14:30`（今日または明日の14:30）
- `2026-01-30 14:30`（日付と時刻）
- `30分後`（相対指定）

### Q: 毎回パスが長くて入力が面倒

**A**: エイリアスを設定することをお勧めします（SETUP.md参照）。または、スクリプトをワークスペース直下に移動することも検討してください。

## 🚀 効率的な使い方

### 1. エイリアスの活用

`.zshrc` または `.bash_profile` に追加：

```bash
alias sendmail='cd "/Users/matsunomasaharu/Library/CloudStorage/OneDrive-個人用/215_神・大家さん倶楽部/C1_cursor/mail_automation" && source venv/bin/activate && python send_mail.py'
```

```bash
sendmail --md-file "メール.md" --excel-file "送信先.xlsx"
```

### 2. よく使うコマンドをメモに保存

頻繁に使うコマンドをテキストファイルに保存しておくと便利です。

複雑なオプションを指定する場合は、Cursorのチャットで自然言語で指示すると、AIが適切なコマンドを組み立ててくれます。
