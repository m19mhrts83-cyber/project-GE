# G2 空室対策メール送信の実行手順

## 重要：Cursor のターミナルでは動きません

**Cursor のターミナル**（Cursor 下部のターミナルパネル）では、Python 対話モード（>>>）になってしまいます。**必ず下記の方法で実行してください。**

---

## 方法1：AppleScript で実行（推奨）

1. **Cursor を終了する**（⌘ + Q）

2. **Finder** を開き、次のフォルダへ移動する:  
   `OneDrive-個人用` → `215_神・大家さん倶楽部` → `C1_cursor` → `mail_automation`

3. **送信_G2_260213.scpt** を **ダブルクリック**  
   → Terminal.app が開き、送信処理が始まる

4. 初回は「開発元を確認できないため開けません」と出る場合がある  
   → 右クリック → **「開く」** を選ぶ

5. ブラウザが開いたら Gmail 認証を完了する

6. 送信完了後、「Enter キーを押すと閉じます...」と出たら Enter を押す

---

## 方法2：Mac のターミナルで直接実行

1. **Cursor を終了する**（⌘ + Q）

2. **Spotlight**（⌘ + Space）で「**ターミナル**」と入力して Enter  
   → **Mac のターミナル**（Cursor ではない）が開く

3. 次のコマンドをコピーして貼り付け、Enter を押す:

```bash
bash "/Users/matsunomasaharu/Library/CloudStorage/OneDrive-個人用/215_神・大家さん倶楽部/C1_cursor/mail_automation/送信_G2_260213.sh"
```

4. ブラウザが開いたら Gmail 認証を完了する

---

## 見分け方

- **Mac のターミナル**: ウィンドウのタイトルが「ターミナル」や「bash」のみ  
- **Cursor のターミナル**: タイトルに「Python」と出る → 対話モードになるため使わない
