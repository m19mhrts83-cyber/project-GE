# メールアドレス一覧の準備方法

メール自動送信システムで使用するメールアドレス一覧のExcelファイルの準備方法を説明します。

## 📋 準備方法

### 方法1: 既存のExcelファイルを使用

既にお持ちの業者一覧などのExcelファイル（例: `★各種リスト.xlsx`）に、メールアドレス列があればそのまま使用できます。

必要な条件:
- メールアドレスを含む列がある
- 列名に「メール」「email」「mail」などの文字が含まれている（自動検出）

### 方法2: 新規にExcelファイルを作成

以下の形式でExcelファイルを作成してください。

## 📊 推奨フォーマット

### 基本形式

| 会社名 | 担当者 | メールアドレス | 備考 |
|--------|--------|----------------|------|
| A不動産 | 田中様 | tanaka@example.com | |
| B管理 | 鈴木様 | suzuki@example.com | |
| C仲介 | 佐藤様 | sato@example.com | |

### 必須項目

- **メールアドレス列**: 必須
  - 列名の例: 「メールアドレス」「Email」「メール」「mail」など
  - 各セルに有効なメールアドレス（@を含む）を記入

### 任意項目

- **会社名**: 送信先の会社名（管理に便利）
- **担当者**: 担当者名（管理に便利）
- **備考**: メモや注意事項（送信には使用されません）

## 🎯 具体的な作成手順

### Excelで新規作成する場合

1. Microsoft Excelまたは互換ソフトを開く
2. 新しいワークブックを作成
3. 1行目にヘッダーを入力:
   - A1: 会社名
   - B1: 担当者
   - C1: メールアドレス
   - D1: 備考
4. 2行目以降に送信先のデータを入力
5. ファイルを保存（例: `送信先メールアドレス一覧.xlsx`）

### Google スプレッドシートで作成する場合

1. Google スプレッドシートで新規シートを作成
2. 上記と同じ形式でデータを入力
3. 「ファイル」→「ダウンロード」→「Microsoft Excel (.xlsx)」を選択
4. ダウンロードしたファイルを使用

## 🔍 複数シートを使う場合

1つのExcelファイルに複数のシートを作成して、用途別に送信先を管理できます：

```
送信先メールアドレス一覧.xlsx
├── 名古屋市北区  ← 志賀本通物件用
├── 名古屋市緑区  ← キャラメル物件用
└── 全業者        ← すべての業者
```

使用時にシート名を指定：
```bash
python3 send_mail.py --md-file メール.md --excel-file 送信先メールアドレス一覧.xlsx --sheet-name "名古屋市北区"
```

## ⚠️ 注意事項

### メールアドレスの確認

- 送信前に必ずメールアドレスが正しいことを確認してください
- タイプミスや古いアドレスは削除または更新してください
- 空白行は自動的にスキップされます

### テスト送信

初めて使用する場合は、少数のアドレス（1〜3件）でテスト送信することを強く推奨します：

1. テスト用のシートを作成（例: 「テスト」シート）
2. 自分のメールアドレスを2〜3件入力
3. テスト送信を実行
4. 正しく送信されることを確認してから本番実行

### セキュリティ

- メールアドレス一覧は個人情報を含むため、取り扱いに注意してください
- Gitなどのバージョン管理システムにはアップロードしないよう推奨します

### .gitignoreへの追加（推奨）

```gitignore
# メールアドレス一覧（個人情報保護）
送信先メールアドレス一覧.xlsx
C1_cursor/mail_automation/送信先*.xlsx
```

## 📞 よくある質問

### Q1: メールアドレス列が見つからないというエラーが出る

**A**: 以下を確認してください：
- 1行目にヘッダー（列名）が入力されているか
- 列名に「メール」「email」「mail」などの文字が含まれているか
- 含まれていない場合は `--email-column "列名"` で明示的に指定

### Q2: 一部のメールアドレスだけに送りたい

**A**: 以下の方法があります：
1. 別のシートを作成して、送信したいアドレスだけをコピー
2. 元のシートで、送信したくない行を削除または別の列に移動
3. 送信後に元に戻す

### Q3: CSV形式でも使えますか？

**A**: 現在はExcel形式（.xlsx）のみ対応しています。CSV形式の場合は、Excelで開いて.xlsx形式で保存してください。

### Q4: 最大何件まで送信できますか？

**A**: 
- 技術的には制限なし
- ただし、Gmail APIの送信制限（個人アカウント: 1日500通）に注意
- 大量送信の場合は、複数日に分けることを推奨
