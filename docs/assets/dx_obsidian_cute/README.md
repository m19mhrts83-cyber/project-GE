# dx_obsidian_cute

`Obsidian紹介_DX互助会向け_cute.html` 用の **手描き風 SVG イラスト**（白背景・細い線・淡色の塗り）。

## 差し替え

- **PNG にしたい場合**: 同じファイル名で `s2.png` 等を置き、HTML の `src` を `.png` に変更するか、`.svg` を上書き export する。
- **画像生成 AI を使う場合**: `docs/_drafts/slides_outline_obsidian_cute.md` の「図案」列をプロンプトにし、出力をここに保存。
