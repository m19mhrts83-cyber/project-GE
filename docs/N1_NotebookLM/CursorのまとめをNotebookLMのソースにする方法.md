# Cursor でまとめた情報を NotebookLM のソースにする方法（初心者向け）

Cursor で作成・まとめたテキストを、NotebookLM の「ソース」（AI が参照する資料）として登録する手順です。NotebookLM をまだ使ったことがない方向けに、画面の流れも含めて説明します。

---

## 1. NotebookLM とは（30秒で理解）

- **NotebookLM** = Google が提供する「**あなたが入れた資料だけ**を根拠に質問に答えてくれる」AI ツールです。
- 「ノート」という単位で、**ソース（資料）** を集めます。PDF・Web・YouTube・**テキストの貼り付け**などがソースになれます。
- チャットで「このノートのソースを踏まえて教えて」と聞くと、**その資料の内容だけ**から回答してくれます（嘘をつきにくい＝ハルシネーションが少ない）。

**今回やること**: Cursor でまとめた文章を、その「ソース」のひとつとして NotebookLM に登録する、です。

---

## 2. 一番簡単な方法：**コピー＆貼り付け**

Cursor でまとめたテキストを **そのままコピーして NotebookLM に貼り付ける** のが、いちばん手軽です。ファイル保存も不要です。

### 手順（ステップごと）

#### ステップ1: NotebookLM を開く

1. ブラウザで **https://notebooklm.google.com** を開く。
2. Google アカウントでログインする（まだの場合）。

#### ステップ2: ノートを作る（初回だけ）

1. 画面の **「新しいノート」** または **「＋」** のようなボタンをクリックする。
2. ノートに名前をつける（例：「Cursorまとめ_税理士比較」）。あとから変更できます。

#### ステップ3: ソースを「貼り付け」で追加する

1. ノートを開いた状態で、左側の **「ソース」** パネルを確認する。
2. **「追加」** または **「ソースを追加」** ボタンをクリックする。
3. 追加方法の一覧が出るので、**「テキストを貼り付け」** や **「Paste text」** のような項目を選ぶ。
   - メニューによっては「テキスト」「貼り付け」などと書いてあります。
4. **Cursor でまとめた文章をすべて選択 → コピー（Cmd+C）** して、表示された入力欄に **貼り付け（Cmd+V）** する。
5. **タイトル** を入力する（例：「法人税理士比較のまとめ」）。空欄でも追加はできますが、あとで見分けやすくなります。
6. **保存・追加** のようなボタンを押す。

これで「Cursor でまとめた情報」が、そのノートのソースの 1 つとして登録された状態になります。チャットで「このソースの内容を要約して」などと聞けば、そのテキストを根拠に答えてくれます。

## 3. 別の方法（用途に合わせて）

### 方法A: ファイルでアップロードする（複数ドキュメントをまとめて入れたいとき）

- Cursor でまとめた内容を **Markdown（.md）やテキスト（.txt）で保存** する。
- NotebookLM の **「追加」→「ファイルをアップロード」** で、そのファイルを選ぶ。
- **ドラッグ＆ドロップ** にも対応しています。
- メリット: 同じフォルダの .md を何本もまとめてノートに入れやすい。Cursor 側のファイルと内容をそろえておける。

### 方法B: Google ドキュメント経由（あとから編集したいとき）

1. **Google ドキュメント**（docs.google.com）で新規ドキュメントを作成する。
2. Cursor でまとめたテキストをコピーして、ドキュメントに貼り付ける。
3. ドキュメントにタイトルをつけて保存する。
4. NotebookLM の **「追加」→「Google ドライブ」** などから、そのドキュメントを選んでソースに追加する。
- メリット: あとから Google ドキュメントを編集すると、NotebookLM 側で「同期」すれば内容を更新できる（NotebookLM のソースパネルに「Google ドライブと同期」が出る場合）。

## 4. よくある疑問

**Q. 1 つのノートに、Cursor のまとめを何本も入れてもいい？**  
A. はい。ソースは最大 50 個まで入れられます。同じノートに「まとめ1」「まとめ2」…と複数追加して、まとめて質問できます。

**Q. 貼り付けるテキストの量に制限はある？**  
A. 1 ソースあたり最大 500,000 語まで、という目安があります。通常の「Cursor でまとめたメモ」程度なら問題ありません。

**Q. 既存のノートに、あとから Cursor のまとめを 1 つ追加したい。**  
A. そのノートを開いた状態で、同じように「追加」→「テキストを貼り付け」で追加すれば大丈夫です。

**Q. Cursor のチャットの回答をそのまま入れていい？**  
A. はい。チャットの返答部分を選択してコピーし、NotebookLM の「テキストを貼り付け」に貼り付ければ、それが 1 つのソースになります。

---

## 5. おすすめの流れ（まとめ）

| やりたいこと | おすすめの方法 |
|--------------|----------------|
| とにかくいちばん簡単に 1 回入れたい | **コピー＆貼り付け**（本文の手順） |
| 複数の .md をまとめてノートに入れたい | **ファイルでアップロード**（.md / .txt） |
| あとから内容を直して、NotebookLM も更新したい | **Google ドキュメントに貼り付け → ドライブから追加** |

まずは **「コピー＆貼り付け」** で 1 本入れてみて、NotebookLM のチャットで「このソースの要点を教えて」と試すと、動きがつかみやすいです。
