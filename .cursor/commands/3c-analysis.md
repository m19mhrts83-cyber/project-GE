# 分析指示

## Customer（顧客）分析
以下の観点で顧客環境を分析してください。
- 顧客が求めている価値・ニーズ
- 顧客の購買行動の特徴
- 顧客のニーズの変化や新たなトレンド
- 市場規模や成長性の傾向

## Competitor（競合）分析
ここまでの調査結果をもとに、以下の観点で分析してください。
- 各競合の戦略の特徴（どこで勝負しているか）
- 競合の強みと弱み
- 競合がカバーできていない領域（空白地帯）
- 競合の今後の動きの予測

## Company（自社）分析
上記の自社情報と、顧客・競合の分析結果を踏まえて分析してください。
- 自社の強み（競合に対して優位な点）
- 自社の弱み（競合に劣る点・改善が必要な点）
- 自社がまだ活かしきれていないリソースや機会

## 統合分析
3つのCを統合し、以下を導き出してください。
- 「顧客が求めていて、競合が提供しておらず、自社が提供できるもの」（勝ちポジション）
- 自社が取るべき戦略の方向性（3つ以内）
- 優先的に取り組むべきアクション（3つ以内）

# 出力ルール
- 事実に基づく分析と、分析に基づく提案を区別してください
- 確認できない情報に基づく推測は「推測」と明記してください
