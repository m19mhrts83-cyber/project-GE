# DX互助会 共有フォルダ構成

**作成日**: 2026-01-31  
**目的**: DX互助会メンバーでアウトプットを共有するGoogle Driveフォルダの構成とルール

---

## 🎯 基本方針

- **共有対象**: メンバー間で共有する成果物・ナレッジのみ
- **個人アウトプット**: 各自のローカル環境で管理（共有する必要があるもののみアップロード）
- **プラットフォーム**: Google Drive（無料、使いやすい、協働編集可能）

---

## 📁 フォルダ構造

```

├─ 00_inbox/              # 仮置き場
├─ 01_prompts/            # テンプレート資産
├─ 02_inputs/             # 素材置き場
├─ 03_outputs/            # AI生成物

├─ 05_knowledge/          # ナレッジ（編集済み）
├─ 06_projects/           # 案件別
├─ 99_archive/            # アーカイブ
└─ README.md              # 使い方ガイド
```

## 📖 各フォルダの詳細説明

### 00_inbox/ - 仮置き場

**用途**: とりあえずアップロードする場所

- まだ整理できていないファイル
- 他メンバーと相談したいもの
- どこに入れるか迷うもの

**ファイル例**:
```
00_inbox/
├─ 260131_松野_メモ.md
├─ 参考になりそうな記事.pdf
└─ 相談したいスクリプト.py
```

### 📤 アップロード時のルール

**1. ファイル名の命名規則**
```

## 🤝 個人アウトプットの扱い

**基本方針**:
- 個人の作業ファイルは**各自のローカル環境**で管理
- Google Driveにアップするのは**共有する必要があるもののみ**

**共有すべきもの**:
- ✅ 完成したスクリプト（他メンバーも使えるもの）
- ✅ 成功したプロンプト（再利用価値があるもの）

**共有不要なもの**:
- ❌ 個人的な試行錯誤のメモ
- ❌ 作業途中のファイル（完成したらアップ）
- ❌ 個人専用のツール設定
- ❌ プライベートな情報を含むもの

## 🚀 セットアップ手順（所要時間: 5-10分）

### Step 1: Google Driveでメインフォルダ作成

1. https://drive.google.com にアクセス
2. 左上の「新規」ボタンをクリック
3. 「新しいフォルダ」を選択
4. フォルダ名: `DX互助会_共有アウトプット`
5. 「作成」をクリック

---

### Step 2: サブフォルダ作成

作成したフォルダ内で、以下の8つのフォルダを作成:

```
✅ 00_inbox
✅ 01_prompts
✅ 02_inputs
✅ 03_outputs
✅ 04_logs
✅ 05_knowledge
✅ 06_projects
✅ 99_archive
```

**効率的な作成方法**:
1. メインフォルダを開く
2. 「新規」→「新しいフォルダ」を8回繰り返す
3. 各フォルダ名を上記の通りに設定

---

### Step 3: README 作成

**方法1: Google Docsで作成**
1. メインフォルダ内で「新規」→「Google ドキュメント」
2. 上記の「README.md の内容」をコピー&ペースト
3. ファイル名を `README` に変更
4. 完了

**方法2: テキストファイルをアップロード**
1. ローカルで `README.md` を作成
2. 上記の内容を保存
3. Google Driveにドラッグ&ドロップ

---

### Step 4: 共有設定

**オプションA: リンク共有（簡単・おすすめ）**

1. `DX互助会_共有アウトプット` フォルダを右クリック
2. 「共有」→「リンクを取得」をクリック
3. 「制限付き」を「リンクを知っている全員」に変更
4. 権限を「閲覧者」→「編集者」に変更
5. 「リンクをコピー」をクリック
6. メンバーにリンクを送信（LINEやメールで）

**オプションB: メールアドレス指定**

1. フォルダを右クリック → 「共有」
2. 「ユーザーやグループを追加」にメンバーのGmailアドレスを入力
3. 権限を「編集者」に設定
4. 「送信」をクリック

**どちらを選ぶ?**
- **リンク共有**: 手軽、新メンバー追加が簡単
- **メール指定**: セキュリティ高い、メンバー管理がしやすい

---

### Step 5: 初回アップロード（テスト）

**今日作った勉強会資料をアップロード**:

1. フォルダ構造を作成:
   ```
   03_outputs/
   └─ 勉強会資料/

   - `DX互助会_第1回勉強会_配布資料.html`（PDF版）

3. できれば作成プロセスも記録:
   - `260131_作成プロセス.md`（どうやって作ったか）

## 📝 議論の記録

### Q&A

**Q: 個人のGoogleアカウントで管理 vs 専用アカウント作成?**

A: 最初は個人アカウントで管理がおすすめ

**理由**:
- セットアップが簡単（5分で完了）
- Googleの無料枠（15GB）で十分
- 管理の手間が少ない
- すぐに始められる

**専用アカウントが必要なケース**:
- 容量が15GBを超える（動画や大量のファイル）
- オーナー権限を特定個人に依存させたくない
- より公式な組織としての管理が必要

→ 最初は個人アカウントで開始、必要に応じて専用アカウントに移行
