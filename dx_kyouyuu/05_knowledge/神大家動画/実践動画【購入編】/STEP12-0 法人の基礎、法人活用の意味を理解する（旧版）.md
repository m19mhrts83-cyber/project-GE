## 法人化のメリット

| メリット | 詳細 |
|---|---|
| 税率の違い | 個人：累進課税で最大55%程度 → 法人：15〜24%程度（新規設立法人） |
| 債務を法人名義で作れる | 個人債務は1〜2億円で融資控え。法人債務は賃貸経営の人格として評価され融資が伸びやすい |
| 経費にできるものが多い | ただしGPKではあまり違いがない |

## デッドクロスの問題

### 構造
- **現金フローと税務上の利益の乖離**：元本返済は経費にならない
- 例：現金は500万円残っているのに、税務上の利益は1,400万円になるケース

### 年数経過による悪化
- 金利返済が減り元本返済が増えると、経費にできる金額が減少
- 税率43%の場合、税引き後の手残りが**33万円**まで減少する可能性

---

## デッドクロス対策

| 対策 | 内容 |
|---|---|
| **対策1：返済期間の調整** | 返済期間10年や15年の短い物件、築古アパート（7年程度）を組み込む。返済を終わらせることで現金増 |
| **対策2：減価償却の調整** | 減価償却期間をできるだけ長くする。4年で償却し終わらないように注意 |
| **対策3：修繕費の活用** | 資本的支出と修繕費の区分けを理解。リフォームローンで現金を減らさずに経費を作る |
| **対策4：戸建て売却益の活用** | 毎年1個戸建てを買って長期譲渡まで持ち込む。短期譲渡の売却益で税金を払う方法も |
| **対策5：法人化による税率引き下げ** | 不動産を法人に移転、または最初から法人で購入。43%の税率を抑えてデッドクロスの影響を軽減 |

## 法人名義での債務と複数法人戦略

- すべて法人で所有すれば個人債務はゼロとなり、金融機関の評価が高くなる
- **魔法陣・法人隠し**：新設法人を作って物件購入→別の新設法人で申し込む手法（過去に流行）
- 現在はこの手法は金融機関に知られてしまい、あまり使えない
- **推奨**：個人で3〜5棟所有後、適切なタイミングで法人を作って育てていくのが一般的

## 法人への資金投入方法

### 1. 資本金の増資
- 設立時に300万円程度あると望ましい（最低1円でも可能）
- 後から増資すると登記費用がかかるため非効率

### 2. 個人から法人への貸付（役員貸付）
- いつでも戻せる
- **重要**：逆の「役員借入」（法人→個人）は**絶対に決算期末に残してはいけない**
- 印象が非常に悪く、金融機関の評価に悪影響
- 決算の3ヶ月前に税理士と打ち合わせして調整する

### 3. 第三者割当増資
- 株式を分割して投資を受ける方法
- 持分が上がると経営に口出しされたり乗っ取られる危険があるため**非推奨**

## 法人から個人への資金移動方法

### 1. 役員報酬
- 決算時期に翌期1年分を決定、毎月支払い
- 毎月変更できない制約
- 社会保険料が**約30%**かかる
- 個人の所得は累進課税の対象
- 役員報酬を出すと平均所得が高くなるため、すぐに出すのは避けたい

### 2. 退職金
- 家族を役員にして5年〜10年後に退職させ、退職金を支払う
- 退職金は**ほぼ経費**
- 中小企業退職金共済（中退共）で**800万円**まで損金で積み立て可能
- 退職時に800万円を法人に戻し、退職金として支払いでプラマイゼロ
- **デメリット**：積立期間中は現金が再投資に回せない、退職させる必要がある

### 3. 生命保険
- 法人で積み立てた生命保険を一定期間後に個人に名義変更
- 例：10年間で累計2,500万円支払い → 10年目に1,500万円で個人に譲渡
  - 法人は1,000万円の損金を計上
  - 個人は11年目に解約して2,500万円の返戻金
  - 一時所得：(1,000万円 - 50万円) ÷ 2 = 475万円が課税対象
  - 実質的に個人に**約850万円**のお金を移せる
- **デメリット**：10年などの期間管理が重要、譲渡時にまとまったお金が必要

## 設立費用と維持費

| 項目 | 費用 |
|---|---|
| 合同会社 設立 | 約10万円（登録費用6万円含む） |
| 株式会社 設立 | それ以上 |
| 税理士費用 | 月1〜3万円程度 |
| 決算費用 | 10〜20万円 |
| **年間維持費合計** | **約50万円** |

> **推奨**：法人化する際は必ず税理士をつけるべき

## その他の戦略

### 管理委託契約
- 個人で物件を買っている間に、その物件の管理委託契約を法人と結ぶ
- 個人の家賃の**8%程度**を法人の売上にできる
- 法人はまだ物件を持っていないが黒字を作れる
- 決算期変更を利用すれば**半年程度で黒字法人**を作れる

### 本業の利益が安定している場合
- 最初から法人で物件を購入
- 個人から法人への出資または貸付で資金を準備
- 本業の子会社として作り、親会社から出資する方法もある

## 重要な注意点

- いつどれくらい戻したいのか、規模拡大をどこまでするのか、退職が可能かなど、**人によって正解が異なる**
- 基本的な知識を理解した上で、個別にプランを組む必要がある
- 具体的な設立や税務相談は**司法書士や税理士**などの専門家に依頼すること
