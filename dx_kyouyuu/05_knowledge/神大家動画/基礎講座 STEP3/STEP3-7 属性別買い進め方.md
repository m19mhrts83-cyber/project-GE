## 主な融資基準と地域別の特徴

### 静岡銀行（静銀）

| 項目 | 条件 |
|---|---|
| 対象エリア | 国道16号内（東京・神奈川はより広域OK）※関東の人のみ |
| 年収条件 | 700万円以上が必要（500万円台では80%程度の融資率） |
| 預貯金条件 | 決済後1,000万円以上、または借入総額の10%以上 |
| 法人融資 | 可能（法人の給与年収1,000万円が好ましい） |
| 融資上限 | 最大209%・35年融資などの長期融資が可能 |

### 滋賀銀行【重要・動画視聴必須】

| 項目 | 条件 |
|---|---|
| 対象 | **全国の方、全国の物件が対象** |
| 融資枠 | ご自身・奥様それぞれ**4500万円まで** |
| 名義 | **個人名義** |
| 融資額 | 路線価の6〜7掛け ＋ 属性枠 → **MAXでオーバーローン** |
| 条件 | 賃貸年収 ＜ 個人のその他年収合計（役員報酬・事業所得も合算可能） |
| 特徴 | 「独占枠」あり、一度に複数物件に融資可能 |

- 配偶者名義での融資も可能で、共同名義や連帯保証人でより柔軟な対応
- **オーバーローンが合法的に出る神手段** → 必ず積極的にプランに入れる

### 香川銀行・徳島大正銀行

- 同じグループで審査基準も同様
- 土地割合を厳格に見ない傾向 → **初心者に向いている**
- アパート投資では10個程度まで対応可能（それ以上は審査が厳しい）
- 新宿支店と中央支店で融資相談可能
- 物件評価と属性が伸びれば**フルローンの可能性あり**（2023/1〜）

### Wix（ウィックス）

- 融資金額が**年収の10倍まで**
- 法人名義でも購入可能

## 投資戦略とプランニング

- 新築物件は比較的融資が通りやすく、中古は特殊性があるため審査が異なる
- 複数の銀行を組み合わせることで価値評価の差が生まれ、有利な条件を引き出せる
- 資金計画として**預貯金2000万円が目安**（状況によって優先順位は変わる）
- 地域によって使える銀行や融資条件が異なるため、地域特性を考慮したプランニングが重要

---

## 効果的な戦略構築のポイント

- 配偶者との共同名義や保証などを活用して**より多くの融資枠を確保**
- 地域特性に合わせた銀行選びが重要
- 個人の状況によって最適な順序や組み合わせは異なるため、**個別のプランニングが必要**
- プランニングによって最終的な投資効果が決まるため、専門家に相談するのが望ましい

---

## 注意点とリスク管理

- 複数物件所有の場合、個人でも法人でも**10個以上になると融資が厳しくなる**
- 序盤の投資戦略が重要で、最初のステップで融資条件が大きく変わる可能性がある

- 静銀は関東の人のみ。関東以外の方はご自身のエリアの地銀等を活用
