## 銀行の評価と審査の仕組み

- 金融機関は借り手に対して**ポイント制の評価システム**を使用
- 評価は3つに分類：

| 評価区分 | 内容 |
|---------|------|
| **定量評価** | 数値で評価できる要素（売上、利益率、自己資本比率など） |
| **定性評価** | 人間性や事業の将来性など |
| **実体評価** | 実際の事業運営状況 |

- 銀行は企業の格付けを行い、**総合判断**で融資の可否を決定
- 住宅ローンやアパートローンは、この評価システムを超簡略化したもの

## 経営者・自営業者向け融資戦略

### 基本方針

- **メインバンクにこだわりすぎない**：融資獲得の機会を逃す可能性がある
- **収益物件に融資してくれる金融機関**を見つけることが重要
- 信用金庫では頭金1割程度、20年融資、場合によっては95%融資も可能
- 融資を受ける前に**複数の金融機関との取引を開始**しておくべき

### 評価点を上げる具体的方法

| 方法 | 詳細 |
|------|------|
| 黒字化 | 最低でも2期、できれば**3期連続で黒字**を出す |
| 会計処理の工夫 | 営業利益率を上げる（特別損益の活用など） |
| 役員報酬 | 支払いや自己資金の確保もポイント獲得につながる |
| 修正申告 | 黒字にする |
| 決算期変更 | 黒字化を早める |

> **決算書の黒字化は融資獲得の第一歩**

## 定年退職者向け戦略

- 定年退職者は**サラリーマン属性が使えなくなる**ため、賃貸経営者としての実績作りが必要

### 重要ポイント

| ポイント | 内容 |
|---------|------|
| 定年前の準備 | アパートローンなど使える融資は**使い切っておく** |
| 法人化 | 家族が役員に入る「**家族法人**」が融資期間を伸ばすのに効果的 |
| 実績作り | 不動産売買の経験を積んで実績を作り、それを基に新規融資を獲得 |

---

## アピール戦略

### 企業リサーチ会社への対応

- **帝国データバンク**などの対応を丁寧に行う
- リサーチ会社が金融機関に報告するため、良い印象を与えることが重要

### 金融機関への提出資料

| 資料 | 目的 |
|------|------|
| 会社概要 | 事業内容・実績のアピール |
| 事業計画書 | 将来性のアピール |
| 最新の決算情報 | 定期的に送ることでアピール |
| 経営歴書 | 経営者としての実績証明 |

---

## 実践的なアプローチ

- 黒字化するまでの間は、**トラストや公庫**など使える融資商品を活用する
- 金融機関との**取引実績を積み上げる**ことが重要
- 融資はオーダーメイドであることを理解し、自身の状況に合った金融機関を探す

---

## アクションアイテム

- [ ] 決算書を黒字化し、できれば2-3期連続の黒字を達成する
- [ ] 経営歴書や事業計画書を作成し、金融機関へのアピール材料を準備する
- [ ] 収益物件に積極的な地域の金融機関をリサーチする
- [ ] 帝国データバンクなどの企業リサーチ会社との良好な関係を構築する
- [ ] 経営実績を積み上げながら、新規融資に向けた準備を進める
