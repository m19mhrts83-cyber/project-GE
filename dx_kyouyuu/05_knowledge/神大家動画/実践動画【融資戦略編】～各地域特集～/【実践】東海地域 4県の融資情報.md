# 【実践】東海地域 4県の融資情報

> **出典**: DB_神大家動画 ／ グループ: 実践動画【融資戦略編】～各地域特集～ ／ 日付: 2025-10-06

---

## 調査概要

- 東海地域（**愛知県、岐阜県、静岡県、三重県**）の金融機関に対する詳細な融資条件ヒアリング結果
- 主に不動産投資向けの融資条件、特に「チクフル」（築古物件のフルローン）や「フルローン」の可能性について調査
- 各金融機関の融資姿勢、評価方法、優先条件などを比較

---

## 愛知県の金融機関

### 名古屋銀行本店営業部

- 事業主や総資産家が対象、サラリーマンの収益運用は厳しい
- 頭金必須だが評価が高ければ理論的にフルローン可能
- 不動産賃貸業として本業より不動産収入の方が多いことが前提条件

### 大垣共立（ガマゴリ）★有望

- **非常に融資条件が緩い**
- 築古物件でも20年ローン、フルローン可能性あり
- 強端フルローンもOK、路線価ベースの評価
- 金利が低く、プレローンも補填可能、同時補填OK
- 運転資金も相談可能で、預金担保融資も可能

---

## 岐阜県の金融機関

### ひだしん組★有望

- 最大20年ローン、フルローン可能性あり
- 強端フルローンの可能性もあり、非常に条件が良い
- ただし高山本社・本店業務のため担当者次第

### 大垣共立岐阜支店

- 地主のみ対応、新築でも22年ローン

## 静岡県の金融機関

| 金融機関 | ローン年数 | フルローン | 特徴 |
|----------|----------|-----------|------|
| **静岡銀行名古屋支店** | 35年可能性あり | 出ないが頭金10%程度 | 関東と違い担保掛目70-75%は必須ではない |
| **浜松磐田信用金庫** | 20年可能性あり | 強端フルローン可能性あり | バックファイナンスもOK、実売価格を考慮 |
| **イオ信組名古屋支店** | - | 出る可能性あり | 取引実績重視。毎年4月スタートの人気商品、9ヶ月程で枠が埋まる |
| **清水銀行** | - | - | かなり審査厳格。金融資産保有額、自己資金、賃貸業実績を重視 |

## 三重県の金融機関

### 三十三銀行

- 案件次第で融資可能性あり
- 新築物件に好意的
- 回答は冷たいが実際には融資実績あり

### その他

- 北伊勢上野信金や桑名信金など、条件厳しいところが多い

## 融資申請のベストプラクティス

| ポイント | 内容 |
|----------|------|
| **経歴書の重要性** | 信用金庫・信用組合では経営経験や実績を強調すべき |
| **キャッシュフロー証明** | 物件のCF（50万〜100万円）が出ていることを証明すると有利 |
| **地域貢献の強調** | 「自分のため」ではなく「街のため・地域のため」という動機付けが重要 |
| **属性の見せ方** | サラリーマン投資家は「空き家対策」などのストーリー性が必要 |
| **評価方法の理解** | 路線価×1.25倍×0.7掛けなど、金融機関ごとの評価方法を把握すべき |

## アクションアイテム

- [ ] 各地域の有望金融機関をリストアップし（緑マークの金融機関を優先）、自分の条件に合うところを選定
- [ ] 経歴書や事業計画書を金融機関ごとにカスタマイズ（特に信用金庫・信用組合向け）
- [ ] 静岡銀行など静岡エリアで融資可能な金融機関を活用
- [ ] 複数の金融機関を組み合わせた戦略を検討（例：浜松岩田で運転資金、西部信金でオーバーローン）
- [ ] 必要に応じて金融機関との関係構築のため取引実績を作る
- [ ] 長期的視点で金融機関との関係強化を検討（場合によっては引っ越しも選択肢に）
