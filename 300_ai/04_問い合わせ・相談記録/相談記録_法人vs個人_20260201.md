## 重要な結論

### ❌ 法人での売上受領はNG

**理由：**
オリックス銀行の融資条件では、**「定款に資産管理以外の事業内容が記載されていると、融資が却下される可能性がある」**

#### リスクシナリオ
```
既存法人でAI事業を行った場合：
1. 年1回の決算書提出時に発覚
2. AI研修の売上が300万円規模 = 無視できない規模
3. 銀行から「事業目的外」と指摘
4. 最悪のケース：期限の利益喪失 → 融資の一括返済請求
```

**定款変更も解決にならない：**
- 定款に他事業を追加 = オリックス銀行の融資条件違反
- 融資が継続できなくなる可能性

## 具体的な活動方法

**登録：** 法人名で登録済み（変更不要）

**説明の仕方：**
```
「商工会には会社で登録していますが、
AI研修の紹介は個人事業として取り組んでいます。
法人では不動産管理をやっていますので。

契約は個人名（松野真治）でお願いします」
```

### 2. 顧客への自己紹介

```
「松野真治と申します。
不動産管理会社を経営しておりまして、
その経験からAI活用の重要性を感じ、

会社経営の経験もありますので、
```

**ポイント：**
- 個人名で活動
- 「法人も経営している」= 信頼性アップ
- 契約はすべて個人名

## その他の相談内容

### 1. 広告ガイドライン（バズbiz活用）

**戦略：** ショート動画→フォロワー獲得→説明会誘導

#### 絶対に禁止される表現
- ❌ 「助成金で必ず出ます」「100%もらえます」
- ❌ 「実質タダ」「タダで受講可能」
- ❌ 「75%確定」「すぐに入金されます」

#### 使用可能な表現
- ✅ 「助成金を活用できる可能性があります」
- ✅ 「審査制のため、必ず受給できるわけではありません」

#### 事前承認が必要
- 助成金に言及するショート動画
- 価格や実質負担額を出す投稿
- サービス名やロゴを使う場合

#### 推奨アプローチ
**Phase 1:** 教育コンテンツでフォロワー獲得（承認不要）
- AIの活用方法、DX事例、経営効率化のヒント

**Phase 2:** 具体的なサービス紹介（承認を取る）
- 内容を作成 → 株式会社ライモに事前承認 → 投稿

### 2. 顧客リスト作成

#### 商工会加盟企業
**月曜日のアクション：**
- 商工会事務局に電話・訪問
- 会員名簿の閲覧・提供を依頼
- 従業員5名以上の企業リストを入手
- 会員向けセミナー開催の可能性を相談

#### 不動産関係者
**情報源：**
- メールの送受信履歴
- 名刺（物理・デジタル）
- 契約書・見積書・請求書
- LINEやChatworkのやりとり

**推奨フォーマット：**
```
| 企業名 | 代表者名 | 担当者名 | 電話番号 | メールアドレス | 
| 従業員数 | 業種 | 関係性 | 優先度 | メモ |
```

## 最終結論

### 採用する戦略
**個人事業主（松野真治）としてAI研修代理店を運営**

### 理由
1. オリックス銀行の融資に影響なし
2. 初期費用0円
3. 「法人経営者」という信頼性も活用できる
4. シンプルで分かりやすい
5. 税務リスクなし

### 今後の展開
- 年間300万円規模で安定稼働
- 将来的に年間500万円超なら法人化を検討
- それまでは個人事業主で十分
