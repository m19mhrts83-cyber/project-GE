# 調査レポート：競合調査（AIリスキリング・導入支援）

## 自社情報
*   **業種**：AIリスキリング教育
*   **主要サービス・商品**：生成AI導入支援（AIリスキリング教育販売）
*   **ターゲット顧客**：中小企業の経営者
*   **対象エリア**：全国

## 競合企業リスト

### 直接競合（5社）
同じ「AIリスキリング・導入支援」を「中小企業・法人」向けに提供している企業です。

| 企業名 | 主要サービス | 競合と判断した理由 |
| :--- | :--- | :--- |
| **株式会社Shift AI** | 法人向けリスキリング支援「SHIFT AI for Biz」、国内最大級AI活用コミュニティ運営 | 「日本をAI先進国に」を掲げ、コミュニティと法人研修の両軸で展開。経営者層への訴求が強く、ターゲットが完全に重複するため。 |
| **株式会社AVILEN** | 生成AI人材育成研修、AI開発・実装支援 | 上場企業としての信頼性と、職種別・階層別のきめ細かい研修カリキュラムを持つ。E資格認定校でもあり、本格的な人材育成ニーズで競合する。 |
| **株式会社キカガク** | DX/AI人材育成研修、eラーニングプラットフォーム「KIKAGAKU」 | AI教育の老舗で、助成金活用（第四次産業革命スキル習得講座など）に強みを持つ。実務直結型の研修で中小企業の導入ハードルを下げている。 |

| **ホリエモンAI学校** | ホリエモンAI学校（オンラインスクール）、法人向け研修 | 堀江貴文氏の知名度を活かし、非エンジニアや経営者層をターゲットにしている。「難しい技術より活用」という切り口が、中小経営者のニーズと合致する。 |

### 間接競合（3社）
異なるアプローチ（安価な自習型、ツール直接契約など）で、顧客の「AIを活用したい」というニーズを満たす企業・サービスです。

| 企業名 | 主要サービス | 競合と判断した理由 |
| :--- | :--- | :--- |
| **Schoo for Business** | 法人向けオンライン研修・動画学習プラットフォーム | 月額定額で「広く浅く」学べるため、コストを抑えてとりあえずAIの基礎を知りたい層が流れる可能性がある。 |
| **Udemy Business** | 法人向けオンライン学習プラットフォーム（講座買い切り・定額） | 実務的な講座が豊富で、特定のスキル（プロンプトエンジニアリングなど）だけをピンポイントで安価に学びたいニーズを満たすため。 |
| **OpenAI / Anthropic** | ChatGPT Plus / Claude Pro（AIツールそのもの） | 「教育を受ける」のではなく、「とりあえずツールを契約して自力で使う」という選択をする層にとっての代替手段となるため。 |
