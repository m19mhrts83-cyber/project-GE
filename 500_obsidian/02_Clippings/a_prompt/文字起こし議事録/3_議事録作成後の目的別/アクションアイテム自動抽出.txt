
プロンプトの目的
文字起こしデータまたは議事録から、担当者・タスク・期限・優先度を表形式で抽出する。
明示的な依頼だけでなく、暗黙の合意もTODOとして拾う。

添付したファイルは、{会議名}の文字起こしテキストです。

この会議で発生したアクションアイテム（やるべきこと）をすべて抽出してください。

【抽出ルール】
- 明示的な依頼（「〇〇さん、これお願いします」）だけでなく、暗黙の合意
（「じゃあ確認しておきます」「来週までに考えます」）もTODOとして抽出する
- 担当者が不明確なものは「担当者未定（要確認）」とする
- 期限が不明確なものは「期限未定（要確認）」とする

【出力形式】
■ アクションアイテム一覧
| No. | タスク内容 | 担当者 | 期限 | 優先度 |
|-----|-----------|--------|------|--------|
|     |           |        |      |        |

■ 要確認リスト（担当者または期限が不明確なもの）
| No. | タスク内容 | 不明確な点 | 該当する発言 |
|-----|-----------|-----------|-------------|
|     |           |           |             |

参加者：{参加者}
