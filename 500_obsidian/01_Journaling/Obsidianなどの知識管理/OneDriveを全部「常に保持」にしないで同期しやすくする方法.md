# OneDrive を「常に保持」にしないで同期しやすくする方法

フォルダ全体のサイズが大きく、OneDrive の **「常にこのデバイスに保持」** を全体にかけられない場合でも、**必要なフォルダだけ** 固定したり、**アプリのアクセスを減らす** ことで、Cursor や Obsidian が読みやすくなります。

---

## 1. 使うフォルダだけ「常に保持」にする（おすすめ）

OneDrive 全体ではなく、**Cursor や Obsidian でよく開くフォルダだけ** を「常にこのデバイスに保持」にします。

**手順**：

1. **Finder** で **OneDrive-個人用** を開く。
2. **215_神・大家さん倶楽部** を右クリック → **「常にこのデバイスに保持」** を選ぶ。
3. **500_Obsidian** を右クリック → **「常にこのデバイスに保持」** を選ぶ。
4. それ以外のフォルダは **「オンラインのみ」** のままでよい。

これで、215 と 500 だけローカルに置き、他はクラウドのままにできます。容量は 215＋500 の分だけ増えます。

---

## 2. Cursor の対象を絞る（.cursorignore）

Cursor が OneDrive 内の **全部のファイル** を読もうとすると、オンラインのみのファイルにもアクセスして同期が重くなります。**編集しないフォルダは Cursor の対象から外す** と、読みに行くファイルが減り、同期負荷が軽くなります。

**各ボルトのフォルダ直下** に `.cursorignore` を置く：

**500_Obsidian 用（500_Obsidian フォルダ直下）**  
`.cursorignore` の中身の例：

```
# Obsidian の設定は Cursor で触らない
.obsidian/
# ワークスペース定義（必要なら編集するので外してもよい）
# *.code-workspace
```

**215_神・大家さん倶楽部用（215 フォルダ直下）**  
編集しないサブフォルダ（例：大量の PDF だけ入っているフォルダ）があれば、そのパスを書く：

```
# 例：このフォルダは Cursor で開かなくてよい
# 50_税金,確定申告/yayoi_import/
```

こうすると、Cursor の検索・インデックスが触る範囲が減り、OneDrive への読み取りが減ります。

---

## 3. Obsidian では「フォルダとして開く」で開き直す習慣

オンラインのみのファイルが多いと、Obsidian の起動時に一斉にアクセスして遅くなることがあります。

- 215 や 500 を開くときは、**「他の保管庫を開く」→ フォルダを指定** で開く。
- よく使うボルトだけ **「常に保持」** にしておき（上記 1）、それ以外はオンラインのままにしておく。

---

## 4. OneDrive の「ファイルオンデマンド」を活かす

- **使うフォルダだけ** 右クリックで「常にこのデバイスに保持」にする（1 の通り）。
- 使わないフォルダは **「オンラインのみ」** のままにしておく。
- 必要なファイルを開いたときにだけ、そのファイルがダウンロードされるので、ディスク容量を節約できます。

---

## 5. 作業するときだけ同期を優先させる

- 215 や 500 で作業し始める前に、**OneDrive のアイコン（メニューバー）** を開き、**「同期の一時停止」** になっていないか確認する。一時停止中なら **「再開」** する。
- 作業中は、215 と 500 以外の大きなフォルダの同期が走ると重くなるので、**「同期の一時停止」** で止めて、作業が終わってから再開する、という使い方もできます（OneDrive のバージョンによっては「特定フォルダだけ同期」などの設定もある）。

---

## まとめ

| やり方 | 効果 |
|--------|------|
| **215 と 500 だけ「常に保持」** | 使うフォルダだけローカルになり、Cursor / Obsidian が読みやすく、同期も軽くなる |
| **.cursorignore で対象を絞る** | Cursor が触るファイルが減り、OneDrive へのアクセスが減る |
| **Obsidian は「フォルダで開く」** | 必要なボルトだけ開けて、起動時の負荷を抑えられる |
| **使うときだけ同期を有効に** | 作業中は同期を止めて重さを避け、あとでまとめて同期できる |

**第一に試すなら**：**215_神・大家さん倶楽部** と **500_Obsidian** の **2 フォルダだけ**「常にこのデバイスに保持」にし、他はオンラインのままにしておく方法が、容量と使いやすさのバランスが取りやすいです。
